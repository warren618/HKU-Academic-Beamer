# HKU Beamer Template — README

This repository provides a clean and minimal Beamer presentation template tailored for academic talks, project presentations, and course reports at **The University of Hong Kong (HKU)**.  
The design emphasizes low-saturation colors, clear structure, and a professional layout with a custom header and footer.

---

## 1. Overview

The template is based on **LaTeX Beamer** and includes:

- A low-saturation HKU-inspired color palette  
- A section navigation bar at the top  
- A three-part custom footline (author — institute — title + page number)  
- Clean frametitles without boxes  
- Section title pages inserted automatically  
- Simple and extensible structure for adding content

Compile using **pdflatex** or **xelatex**.

---

## 2. Features

### ✔ HKU Color Theme
Three core colors are defined:

- `HKUDark` — primary dark teal  
- `HKULight` — light teal for highlights  
- `HKUGrey` — neutral grey  

These colors are applied to titles, structure elements, navigation bars, and the custom footline.
